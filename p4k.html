Python App: Pitchfork Album Reviews

Role: UI/UX Designer, Developer

Using Python, Flask, SQLAlchemy, Spotipy, and Spotify's API

This project requests data about Caribou's playlist, The Longest Mixtape: 1000 Songs for You, from Spotify, via their API, and stores those tracks in a database using SQLAlchemy.

Data was sourced through a pre-existing Github repository containing Pitchfork album review information, in addition to further webscraping conducted with Beautiful Soup.

I organized this information in a database, based on many-to-one relationships.

In order to simplify the relationships between entities in the database, we can assume the following rules:

  • Any single track will have only one album associated with it.
  • Any single album can have many tracks associated with it.
  • Any single album will only have one review associated with it.

Application Flow

Users can check to see if certain songs or artists are in that playlist by entering a title or artist name as part of a Flask route:

  • If a song is in the playlist, the application will also return Pitchfork’s rating of the album the song is present on, if available.
  • If an artist is in the playlist, the application will return the song(s) present by the artist.

The following are screenshots of the program running through various pathways provided by the user.

Route 1

/ → shows you a greeting + gives instructions on how to user should enter information

Route 2:

/check/song/<trackname> → checks if a certain track exists in the database/playlist

Route 3:

/check/artist/<artistname> → checks if tracks by a given artist exist in the database/playlist

Error messages:

To provide error prevention/recovery in the case that incorrect or unusable information is provided.

Future considerations

I'd love to create a version of this script that reversed the general process: a user could search a track title and a list of the user's playlists (owned or followed) containing that track would be returned. I have many playlists and lose track of where I added certain songs :)
